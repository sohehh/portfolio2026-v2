글로벌 유저 타켓의 시술 예약 시스템 및 여러 콘텐츠 페이지를 담고있는 글로벌 서비스 구축 프로젝트입니다.

사용자 페이지의 경우 클라이언트 측에서 제공한 디자인 리소스를 바탕으로 전체 웹 퍼블리싱을 전담했습니다.
시술 상세 페이지는 단순 통이미지 방식이 아닌 텍스트와 이미지를 분리한 시맨틱 마크업으로 작업하여 클라이언트의 요구사항인 SEO최적화를 진행했습니다. 또한 다양한 디바이스 환경에 대응하도록 반응형 웹으로 구현했습니다.
기본적인 UI인터렉션을 구현하여 개발자와의 협업 효율을 높였습니다.
