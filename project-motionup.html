비대면 환경에서 수영 강좌를 진행할 수 있는 웹 기반 플랫폼 구축 프로젝트입니다. 서비스 특성에 맞춰 일반 사용자와 강사라는 두가지 타겟 유저 환경을 구분하여 설계되었습니다.

모바일 해상도를 기준으로 UI/UX 디자인과 웹 퍼블리싱을 전담하여 진행했습니다. 일반 사용자 페이지는 강사 탐색과 수영 피드백 질문이 자연스레 이어지도록 UX를 설계했으며, 강사 페이지도 수강생 질문 관리가 모바일에서도 편리하게 가능하도록 편의성에 초점을 맞춰 작업했습니다.

UI인터렉션은 웹 표준을 준수하는 스크립트로 안정적으로 구현하여 개발 협업 효율을 높였습니다.
